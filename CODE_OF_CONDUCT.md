# Code of Conduct

MXID adopts the **Contributor Covenant, version 2.1** as its community code of conduct.

The full text — including the standards, enforcement responsibilities, scope, and enforcement guidelines — is available at the canonical URL:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>

## Scope

This Code of Conduct applies within all MXID community spaces — GitHub issues, pull requests, discussions, code reviews — and applies when an individual is officially representing the community in public spaces.

## Enforcement contact

Reports of unacceptable behavior may be sent to the project maintainer at **0xkerbos@gmail.com**. All reports are reviewed and investigated promptly and fairly. Reporter identity is kept confidential.

For security vulnerabilities, see [SECURITY.md](SECURITY.md) instead.
